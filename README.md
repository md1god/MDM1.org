# 🏛️ MDM1 — هيكل الموقع الكامل

## 📁 شرح كل مجلد وملف

```
mdm1.org/
│
├── index.html                 ← البوابة الرئيسية + القصة الكاملة
│
├── pages/                     ← كل الصفحات الداخلية
│   ├── hub.html               ← المنصة (سعر + عقد + روابط فقط)
│   ├── whitepaper.html        ← الورقة البيضاء
│   ├── join.html              ← انضم + اشتراك + قاعدة بيانات
│   ├── community.html         ← المجتمع + Wall of Believers
│   ├── roadmap.html           ← خارطة الطريق البصرية
│   ├── tokenomics.html        ← توزيع التوكن + رسم بياني
│   ├── sharia.html            ← التوافق الإسلامي + الأدلة
│   ├── faq.html               ← الأسئلة الشائعة
│   ├── game.html              ← لعبة رع وأنوبيس
│   ├── quiz.html              ← اختبار: على أي جانب أنت؟
│   ├── gallery.html           ← معرض الصور والفيديوهات
│   ├── dashboard.html         ← البيانات الحية (بعد الإطلاق)
│   ├── blog.html              ← المدونة والأخبار
│   └── about.html             ← من نحن + تاريخ المشروع
│
├── css/
│   ├── global.css             ← ✅ مشترك بين كل الصفحات
│   ├── index.css              ← تصميم الواجهة الرئيسية فقط
│   ├── hub.css                ← تصميم المنصة فقط
│   ├── whitepaper.css         ← تصميم الورقة البيضاء فقط
│   └── pages.css              ← تصميم مشترك للصفحات الجديدة
│
├── js/
│   ├── global.js              ← ✅ مشترك بين كل الصفحات
│   ├── index.js               ← منطق الواجهة الرئيسية فقط
│   ├── hub.js                 ← منطق المنصة فقط
│   └── pages.js               ← منطق مشترك للصفحات الجديدة
│
├── images/
│   ├── index/                 ← صور الواجهة الرئيسية فقط
│   ├── hub/                   ← صور المنصة فقط
│   └── shared/                ← صور مشتركة بين الصفحات
│
└── audio/
    ├── background.mp3         ← موسيقى الواجهة الرئيسية
    └── whitepaper.mp3         ← موسيقى الورقة البيضاء
```

## ⚠️ قواعد مهمة

1. **أي صورة** خاصة بصفحة واحدة → ضعها في مجلدها (index/ أو hub/)
2. **أي صورة** تظهر في أكثر من صفحة → ضعها في shared/
3. **navbar و footer** يُكتبان مرة واحدة فقط في global.js
4. **الروابط** من pages/ للـ root تبدأ بـ ../ مثل ../css/global.css
5. **الروابط** من index.html للـ pages تبدأ بـ pages/ مثل pages/hub.html

## 🗺️ ترتيب رحلة المستخدم

```
index → join → whitepaper → tokenomics → sharia → hub → community
```
